    Tremor is now Open Source

    Announcement of Tremor as an OSS project
    1 minute read Published: 22 Feb, 2020

    The tremor project was released as an Open Source software project on 22nd February 2020 at 20:02:02.

    Catch us at BOB2020 on the 28th February 2020 in Berlin where we will be talking about our usage of the Rust programming language to build high performance and high density production systems with Tremor.
